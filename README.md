# ניתוח מבדקי הסמכה לספקים

אפליקציית Web מקומית לניתוח ממצאי מבדקי הסמכה של ספקים. אין צורך בהתקנה — פותחים את `index.html` ב-Chrome או Edge.

## שימוש

### קובץ יחיד (מומלץ לשיתוף)

פתח **`standalone.html`** ישירות ב-Chrome או Edge — הכל מוטמע בקובץ אחד (CSS + JS).

ליצירה מחדש מהמקור:
```bash
node scripts/build-single.mjs
```

### מצב פיתוח (קבצים נפרדים)

1. שים את קבצי המבדק בתיקייה אחת (או בתת-תיקיות לפי ספק).
2. **מומלץ:** הרץ `start-server.bat` ופתח http://localhost:8765 (עובד אמין יותר מפתיחה ישירה של index.html).
3. לחץ **בחר תיקייה** ובחר את התיקייה (לדוגמה `samples`).
4. הניתוח ירוץ **אוטומטית** — עיין בלשוניות: השוואה, פירוט ספקים, דוח מנהלים.

## פתרון בעיות

| בעיה | פתרון |
|------|--------|
| "לא נמצאו ממצאים" | ודא שבחרת תיקייה עם קבצי `.csv`/`.xlsx`/`.docx`/`.pdf` (לא רק `index.html`) |
| הכפתורים לא מגיבים | השתמש ב-`start-server.bat` במקום לפתוח את הקובץ ישירות |
| קובץ נטען אבל 0 ממצאים | בדוק לשונית **קבצים** — שם מופיעה סיבת הכשל לכל קובץ |
| Excel/Word/PDF | ודא שיש עמודות: שם ספק, ממצא, חומרה (Major/Minor) |

## פורמטים נתמכים

| פורמט | סיומת | הערות |
|-------|--------|--------|
| Excel | `.xlsx`, `.xls` | מומלץ — דיוק גבוה |
| CSV | `.csv` | מומלץ |
| Word | `.docx` | טבלאות + טקסט חופשי |
| PDF | `.pdf` | heuristics — דיוק תלוי במבנה |

**לא נתמך:** `.doc` (Word ישן)

## מבנה קובץ מומלך (Excel/CSV)

| עמודה | דוגמה |
|-------|--------|
| שם ספק / Supplier | חברת ABC |
| תאריך מבדק / Audit Date | 2024-03-15 |
| שאלה / Question | 4.2.1 — תיעוד נהלי עבודה |
| **הערכה / Evaluation** | **מתאים · לא מתאים · מינורי · מהותי** |
| הערות / Comments | (אופציונלי) תיאור הממצא |

**שדה הערכה:**
| ערך | משמעות |
|-----|--------|
| מתאים | אין ממצא — לא נספר |
| לא מתאים | ממצא Major |
| מינורי | ממצא Minor |
| מהותי | ממצא Major |

ניתן גם להשתמש במבנה ישן: עמודת ממצא + עמודת חומרה (Major/Minor).

## שם קובץ ותיקייה

אם שם הספק או התאריך חסרים בקובץ, המערכת מנסה לחלץ מהשם:
- `SupplierName_2024-03-15.xlsx`
- תת-תיקייה בשם הספק

## חישוב ציון (ברירת מחדל)

- ציון בסיס: **100**
- ניכוי Major: **-10** לכל ממצא
- ניכוי Minor: **-3** לכל ממצא

| ציון | דירוג |
|------|--------|
| 90–100 | מצוין |
| 75–89 | טוב |
| 55–74 | בינוני |
| 0–54 | חלש |

ניתן לשנות בהגדרות (⚙) — נשמר ב-localStorage.

## תיקיית samples

שים קבצי מבדק לדוגמה ב-`samples/` לבדיקה מקומית.

## מגבלות

- בחירת תיקייה מלאה: Chrome/Edge (File System Access API)
- PDF ו-Word פחות מובנים מ-Excel
- הכל רץ locally — אין שליחת נתונים לשרת חיצוני
